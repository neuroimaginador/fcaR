# Launch the fcaRviz Interactive Viewer

This function opens the interactive Shiny application fcaRviz for visual
exploration.

## Usage

``` r
run_fcaRviz(...)
```

## Arguments

- ...:

  Additional arguments passed to
  [`runApp`](https://rdrr.io/pkg/shiny/man/runApp.html).

## Examples

``` r
if (FALSE) { # \dontrun{
run_fcaRviz()
} # }
```
