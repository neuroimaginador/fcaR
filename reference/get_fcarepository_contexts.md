# Get Metadata from the FCA Repository

Connects to the official FCA Repository (https://fcarepository.org) and
downloads the metadata for all available datasets.

## Usage

``` r
get_fcarepository_contexts()
```

## Value

A list containing the metadata for each context (title, dimensions,
description, source).

## Examples

``` r
if (FALSE) { # \dontrun{
meta <- get_fcarepository_contexts()
} # }
```
